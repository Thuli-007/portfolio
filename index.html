    Event Scheduler Application

    Project Description: The Event Scheduler App is a simple desktop application designed to help users manage and organize their events. The app allows users to add, list, edit, search, and delete events efficiently. Each event can include a title, description, date, and time. The application ensures that events are stored in memory and presented in an easy-to-view list, sorted by their scheduled date and time. The user-friendly interface, built with Tkinter, makes it easy to navigate through various options and manage events with minimal effort.
